# Job Aid: Create and Manage Invoices

Audience: Office/admin users

Goal: Create invoices from completed work, download printable invoices, and record payments.

## Before you start

- Work must be completed in Dispatch, Field Mode, or Add Application.
- The customer should have a Quote entered.
- The application must not already be invoiced.

## Create a draft invoice

1. Open the app.
2. Choose **Office Mode**.
3. Open **Invoicing**.
4. Select **Ready to Invoice**.
5. Choose the customer.
6. Review completed applications.
7. Check the work items to invoice.
8. Confirm **Invoice Date**.
9. Confirm **Due Date**.
10. Add invoice notes if needed.
11. Click **Create Draft Invoice**.

## Review or print an invoice

1. Open **Invoicing**.
2. Select **Invoices**.
3. Choose the invoice status filter.
4. Select the invoice under **Open Invoice**.
5. Review invoice lines.
6. Click **Download Printable Invoice**.
7. Open the downloaded `.html` file.
8. Print from the browser.

## Update invoice status

1. Open **Invoicing → Invoices**.
2. Select the invoice.
3. Choose the new status:
   - Draft
   - Sent
   - Partially Paid
   - Paid
   - Void
4. Click **Save Invoice Status**.

## Record a payment

1. Open **Invoicing → Payments**.
2. Select the invoice.
3. Confirm **Payment Date**.
4. Enter **Amount**.
5. Enter **Payment Method**.
6. Enter a reference number if available.
7. Add notes if needed.
8. Click **Save Payment**.

## Status guide

- **Draft**: Invoice created but not sent.
- **Sent**: Invoice sent to customer.
- **Partially Paid**: Payment received but balance remains.
- **Paid**: Balance is zero.
- **Void**: Invoice should no longer be collected.

## Troubleshooting

- If work does not appear under Ready to Invoice, confirm the application was completed.
- If the amount is zero, check the customer Quote field.
- If an item disappeared from Ready to Invoice, it may already be attached to an invoice.
